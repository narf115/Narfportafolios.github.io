Demostraciónes
Nombre del proyecto Información Fotografias
Tower Defenser Un proyecto realizado durante el 2021, con la finalidad de adaptarme a Unity 2D.
Mecánicas: Sistema de reincio y de empezar, los enemigos siguen una trayectoria y las torretas los detectan y les disparan.
Commando Un proyecto realizado durante el 2022, basado en el juego Commando.
Mecánicas: Sistema de disparos, sistema de vida, temporizador, movimiento de cámara, sistema de particulas y de sonido.
VVVVVV Un proyecto realizado durante el 2022 al finalizar el primer curso de Dam-Vi, proyecto que se realizo para poder comprobar las habilidades adquiridas durante el curso.
Mecánicas: Sistema de coleccionables, cambio de gravedad, guardar y cargar partida, cambio de pantallas y sistema de sonido y particulas.
Tiny Run Un proyecto realizado durante el 2022 (Octubre a Diciembre), con la utilidad de adaptarme al mundo 3D de unity.
Mecánicas: Sistema de animaciones, de vida y movimiento, coleccionables, cambio entre escenas, sistema de teleportación, música y sonido.
Cuphead Un proyecto realizado durante el 2023 (durante el mes de Enero y Febrero), con la finalidad de aprender a usar maquinas de estados para marcar las fases del combate del enemigo.
Mecánicas: Sistema de vida, movimiento y disparos, enemigo con distintas fases mediante maquina de estados.
Uncursed Este proyecto se realizo durante Febrero del 2023 hasta el Junio del mismo año, este proyecto fue realizado entre distintos miembros del curso. Mi parte fue programar el minimapa, los coleccionables, misiones y los sistemas de particulas
The Binding of Isaac Un proyecto realizado durante el verano del 2023, proyecto con la finalidad de replicar el sistema de combate, el sistema de salas y el sistema de items del juego.
Mecánicas: Sistema de disparos, vida, objetos y enemigos.
Unreal 3D Durante el curso, se realizo un proyecto de Unreal 3D, en este proyecto se mezclo blueprints y codigo para su funcionamiento.
Mecánicas: Sistema de movimiento, disparos y vida, Hud.
